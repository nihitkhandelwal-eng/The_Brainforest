### Laptop Research
- Use case
	- Coding
	- Editing, Designing
	- Multitasking
	- Gaming
- Frame → Go with Metal Only
- Size → 15.6 or above
- Display Panel → Go with IPS only
- Processor → AMD
	- R5 3600 or 4600
	- If intel, then i7 or above 
	- Generation 9th gen or above
- RAM → 16 GB DDR4
- GPU → Nvidia
- Storage → SSD
