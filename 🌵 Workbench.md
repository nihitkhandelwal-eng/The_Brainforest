	- Search for text in drawing → Alt + S 
	- Toggle property-panel tray-mode → Alt + Shift + T 
	- Transclude (embed) a drawing → Ctrl+Alt+Shift+D

	- Insert image from the vault → Alt + Shift + I 
	- Insert link to file → Alt + Shift + K
